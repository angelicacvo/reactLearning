# Comandos Esenciales

## Proyecto React

```bash
# Iniciar servidor desarrollo
npm run dev

# Compilar para producción
npm run build

# Ver build de producción
npm run preview

# Instalar dependencias
npm install

# Agregar paquete
npm install nombre-paquete

# Eliminar paquete
npm uninstall nombre-paquete
```

## Git

```bash
# Inicializar repo
git init
git add .
git commit -m "Initial commit"

# Subir a GitHub
git remote add origin URL_DE_TU_REPO
git push -u origin main

# Commits diarios
git add .
git commit -m "Descripción de cambios"
git push
```

## Paquetes Útiles

```bash
# React Router
npm install react-router-dom

# Material-UI
npm install @mui/material @emotion/react @emotion/styled @mui/icons-material

# Axios (alternativa a fetch)
npm install axios

# React Hook Form
npm install react-hook-form

# Zod (validación)
npm install zod

# Date-fns (fechas)
npm install date-fns
```

## Terminal

```bash
# Cambiar directorio
cd ruta/del/proyecto

# Listar archivos
dir         # Windows
ls          # Mac/Linux

# Limpiar terminal
cls         # Windows
clear       # Mac/Linux

# Crear archivo
type nul > archivo.txt      # Windows
touch archivo.txt           # Mac/Linux

# Ver directorio actual
cd          # Windows
pwd         # Mac/Linux
```
